Designkonzept
=================
(http://markdown.de/syntax/index.html)


Servicenav
--------------
Die Servicenav ist ganz oben. Sie ist immer sichtbar und enthält zentrale elemente wie 

 - Small Branding
 - Login
 - Suche
 - Prozessauswahl
 
 
Subnav
------------
Die Subnav ist dynamisch. Sie enhält entweder ein **Big-Branding** oder die **History Naviagation**


Content
------------
Der Content enthält Suchergebnisse oder Formulare


Notes - Home
------------------
Die notes Pages ist eine Sonderseite. Sie enhält vermutlich ein 3 spalties Layout

 - Meine Aufgaben   |  Aufgabe starten   |    Faforiten
 - Meine Vorgänge   |  Administration
